国内镜像快速下载Android源码

By Gong Gaofeng

发表于 2015-09-12

文章目录

由于google官方的下载源不能访问，原因大家都懂得，vpn小水管对于下载Android源代码只能呵呵了。幸好清华大学镜像为我们提供了Android源码的下载源，速度杠杠的，所以心动地立即搞起来。
清华大学镜像源官方介绍
—下载环境
我使用的设备是Macbook Pro＋OS X10.10.5。
1.首先创建区分大小写的磁盘映像，大小30G，因为源码的编译需要大小写敏感的文件系统，而mac默认的文件系统是大小写不敏感的。创建方法：
应用程序->实用工具－>磁盘工具->文件－>新建->空白磁盘映像

2.安装XCode。
3.安装MacPort
4.从MacPorts安装make, git, and GPG packages：
$ POSIXLY_CORRECT=1 sudo port install gmake libsdl git-core gnupg
5安装Git Repo：
$ mkdir ~/bin
$ PATH=~/bin:$PATH
6.curl https://storage.googleapis.com/git-repo-downloads/repo > ~/bin/repo（需要科学上网），科学不了的可以尝试下面的命令：
curl “http://php.webtutor.pl/en/wp-content/uploads/2011/09/repo“ > ~/bin/repo
$ chmod a+x ~/bin/repo
初始化Git Repo进入开始时挂载的镜像硬盘，建立一个工作目录：
$ cd /Volumes/android_source
$ mkdir ANDROID_SOURCE/
$ cd ANDROID_SOURCE/
repo init -u git://aosp.tuna.tsinghua.edu.cn/android/platform/manifest -b android-5.1.1_r3
repo sync
或者
repo sync -j4（四个线程并发下载）
接下来就等待吧。
